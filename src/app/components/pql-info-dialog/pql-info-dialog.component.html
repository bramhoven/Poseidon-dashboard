How to use PQL

PQL is Poseidon Query Language. It is used to query health checks to easily filter servers.

PQL is quite easy to use. You can query any health check data item. You can compare this using one of the following comparisons:

  • Not equal: !=
  • Equals: ==
  • Greater than: >
  • Greater or equal: >=
  • Less than: <
  • Less or equal: <=

You can also use multiple comparisons by using the AND keyword

A few examples:

  • ramAvailable >= 4 AND cpuUsage < 50
  • ramAvailable <= 12
  • ramUsage > 75
